Guía didáctica

Esta guía está dirigida al profesorado y recoge los elementos necesarios para la aplicación del proyecto en el aula, incluyendo la relación entre tareas, criterios de evaluación, metodología y orientaciones didácticas.

Finalidad: Facilitar la implementación del proyecto y garantizar su coherencia pedagógica.

Contexto

  • Ciclo Formativo de Grado Superior
  • 2º curso de Desarrollo de Aplicaciones Web (DAW)
  • Módulo: Diseño de Interfaces Web
  • Unidad: Hojas de estilo avanzadas

Metodología

  • Aprendizaje Basado en Proyectos: el aprendizaje se centra en la creación de un producto real
  • Flipped Classroom: contenidos teóricos fuera del aula
  • Gamificación: uso de herramientas interactivas para reforzar conceptos
Este enfoque favorece un aprendizaje activo, significativo y conectado con el entorno profesional.

Atención a la diversidad (DUA)

  • Diferentes formas de acceso a la información (visual, práctica, interactiva)
  • Trabajo cooperativo
  • Flexibilidad en el desarrollo del proyecto
  • Uso de recursos curados como apoyo

Resolución de problemas técnicos

Durante el desarrollo del proyecto pueden surgir dificultades técnicas. Para afrontarlas:

  • El docente debe anticipar los problemas más frecuentes
  • Es necesario disponer de alternativas (plan B)
  • El alumnado debe aprender a buscar soluciones en:
    • Documentación técnica
    • Foros especializados
    • Buscadores
    • Herramientas de inteligencia artificial
Competencia clave: Saber resolver problemas técnicos de forma autónoma es una habilidad fundamental en el desarrollo profesional.

Uso de inteligencia artificial

  • Uso guiado y supervisado
  • Desarrollo del pensamiento crítico
  • Validación del contenido generado
Importante: La IA debe utilizarse como herramienta de apoyo, no como sustituto del aprendizaje.
