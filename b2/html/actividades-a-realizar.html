Actividades del proyecto

A lo largo del proyecto realizarás distintas actividades que te permitirán adquirir las competencias necesarias para desarrollar el producto final.

Importante: Todas las actividades están orientadas a resolver problemas reales de desarrollo web.

Actividad 1: Debate técnico sobre herramientas

Antes de comenzar el desarrollo del proyecto, analizaremos distintas herramientas utilizadas en el desarrollo frontend.

  • Se llevará a cabo una tarea de investigación para seleccionar las herramientas IA más actuales y relevantes para la programación Frontend,
  • Posteriormente se realizará un debate en clase para seleccionar las mejores herramientas y trabajar con ellas.

Actividad 2: Práctica con Flexbox y Grid

Aprenderás a utilizar sistemas de maquetación mediante herramientas gamificadas.

  • Flexbox Froggy
  • CSS Grid Garden
Estas actividades permiten comprender visualmente cómo funcionan los sistemas de maquetación.

Actividad 3: Desarrollo del proyecto web

  • Diseño de la estructura de la web
  • Implementación con CSS avanzado
  • Adaptación responsive
  • Integración de componentes

Actividad 4: Pruebas y mejora

  • Detección de errores
  • Optimización del diseño
  • Uso de IA para depuración

Actividad 5: Presentación final

  • Exposición del proyecto
  • Demostración funcional
  • Explicación de decisiones técnicas
Consejo: No esperes al final para probar tu web. Itera constantemente.
