Producto final del proyecto

El objetivo de este proyecto es el desarrollo de un sitio web profesional completo y responsivo, funcional y adaptado a diferentes dispositivos, que represente un negocio real o verosímil.

Reto: Diseñar y desarrollar una web que podría existir en el mundo real.

¿Qué debes entregar?

  • Un sitio web completo y funcional
  • Diseño responsive (adaptado a móvil, tablet y escritorio)
  • Estructura clara y navegación coherente
  • Uso de Flexbox y Grid
  • Aplicación de Media Queries

Requisitos técnicos

  • Maquetación moderna mediante CSS avanzado
  • Separación adecuada entre contenido y estilo
  • Uso de buenas prácticas de desarrollo
  • Organización del código clara y mantenible

Uso de Inteligencia Artificial

Se permite el uso de herramientas de IA, pero con condiciones:

  • Debes entender el código generado
  • Debes adaptarlo a tu proyecto
  • No se aceptará código copiado sin comprensión

Entrega del proyecto

  • Repositorio en GitHub correctamente organizado
  • Proyecto funcionando
  • Entrega en el formato indicado

Presentación final

  • Explicación del proyecto
  • Demostración del funcionamiento
  • Justificación de decisiones técnicas
Objetivo real: No solo crear una web, sino desarrollar una forma de trabajar similar a la de un entorno profesional.
