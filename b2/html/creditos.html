Créditos y Descarga de este ABP

Este recurso educativo ha sido desarrollado como parte de una situación de aprendizaje en el módulo de Diseño de Interfaces Web.

Descripción: Proyecto de desarrollo de un sitio web profesional con apoyo de inteligencia artificial, orientado a alumnado de 2º CFGS de Desarrollo de Aplicaciones Web.
