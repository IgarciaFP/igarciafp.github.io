Desarrollo web Responsivo con apoyo de IA

Nivel: 2º CFGS DAW
Módulo: Diseño de Interfaces Web
Duración: 12–15 sesiones

En este proyecto vas a diseñar y desarrollar un sitio web profesional completo para un negocio realista, utilizando técnicas avanzadas de CSS y herramientas actuales de inteligencia artificial.

¿Qué vas a aprender?

  • Diseño responsive con Media Queries
  • Maquetación con Flexbox y Grid
  • Organización visual profesional
  • Uso crítico de herramientas de IA
Importante: No se trata solo de hacer una web que funcione, sino de entender cómo y por qué funciona.

¿Cómo trabajarás?

  • Aprendizaje basado en proyectos (ABP)
  • Trabajo en grupo
  • Uso guiado de inteligencia artificial
  • Actividades prácticas y gamificadas
Ruta del proyecto:
  1. Comprender el reto
  2. Investigar herramientas
  3. Diseñar la web
  4. Desarrollar el proyecto
  5. Publicar y presentar
