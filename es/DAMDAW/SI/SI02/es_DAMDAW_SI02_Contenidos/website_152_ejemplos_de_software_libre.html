1.5.2.- Ejemplos de software libre.

Veamos algunos ejemplos de aplicaciones de software libre:

  • Sistemas Operativos: Debian GNU/Linux, Ubuntu, Linex, Guadalinex, MAX...
  • Entornos de escritorio: GNOME, KDE...
  • Aplicaciones de ofimática: OpenOffice, LibreOffice, KOffice, LATEX...
  • Navegación web: Firefox, Konqueror...
  • Aplicaciones para Internet: Apache, Zope...
  • Gestor de correo: Thunderbird, Evolution...
  • Reproductores multimedia: VLC, MPlayer...
  • Antivirus: ClamWin
