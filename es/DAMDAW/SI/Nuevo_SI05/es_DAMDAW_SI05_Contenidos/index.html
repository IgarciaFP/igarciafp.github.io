    SI05N.- Instalación y configuración de sistemas

    Caso práctico

    María, trabajadora en BK programación ha comentado que tiene un serio problema de seguridad en su ordenador. Lleva bastante tiempo sin actualizar el sistema operativo porque tiene un software que fue desarrollado a medida y que resultaría bastante caro de reemplazar.

    Como suele ocurrir con este tipo de software, no existe una actualización de las librerías de las que depende y por tanto si actualiza el sistema perdería el programa.

    Pedro sugiere una estrategia práctica para solucionar dicho problema: virtualizar el entorno actual.

    María estudia la solución propuesta y concluye que lo mejor sería instalar un sistema dual Windows/Linux y utilizar hipervisor de VirtualBox sobre Windows y Docker sobre Ubuntu.

    Ya que el hardware de su ordenador es un poco justo para la virtualización, quiere además utilizar las versiones más ligeras de Windows 10 y de Ubuntu en su equipo.
