Instalación limpia.

El primer método que vamos a ver es el llamado de Instalación Limpia, es el que se hace sobre una partición nueva o sobre una que ya exista pero que será formateada y perderá todo su contenido.

Lo normal, cuando aún no haya nada instalado en el equipo, será insertar el disco de instalación de Windows 10 en la unidad de DVD y hacer que el ordenador arranque desde ella para que la instalación se inicie de forma automática. Como alternativa se puede iniciar el equipo desde una memoria USB preparada con el software de instalación. Otra opción es arrancar el ordenador desde un sistema previamente instalado para iniciar desde él la instalación del nuevo sistema a partir del DVD de instalación o de una memoria USB.

Una vez iniciada la instalación, por cualquiera de los métodos posibles, sólo hay que seguir el asistente e introducirle la información que necesite según la vaya solicitando. Puede haber pequeñas variaciones de una instalación a otra en base al hardware detectado. Por ejemplo, si no hay tarjetas de red inalámbricas el asistente no mostrará la pantalla en la que se piden sus parámetros de configuración para conectarse a la red y buscar nuevas actualizaciones.
